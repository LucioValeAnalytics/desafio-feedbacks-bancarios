# Desafio Criativo: Extraindo Insights do Feedback de Clientes Bancários

Este repositório contém a entrega do desafio criativo do Bootcamp Bradesco, com foco na criação de um prompt para análise de feedbacks de clientes bancários usando IA.

## Arquivos

* `prompt-principal.pdf`: contém o prompt principal com objetivo, contexto, instruções de análise, formato da resposta e restrições.
* `mini_base.md`: contém uma base fictícia de feedbacks bancários para testar o prompt.

## Objetivo

A proposta é orientar uma IA a analisar feedbacks de clientes bancários sobre aplicativo, Pix, cartão e atendimento, identificando problemas recorrentes, elogios, pontos de atrito e oportunidades de melhoria.

A análise deve apoiar supervisores, encarregados, gerentes e equipes responsáveis por experiência do cliente e canais digitais na tomada de decisão.

## Cuidados considerados

O prompt orienta a IA a usar apenas os dados fornecidos, não inventar conclusões e não expor dados pessoais ou sensíveis.
